# 📧 Email Spam Classifier

An Email Spam Classifier built using Machine Learning and Natural Language Processing (NLP) that classifies emails as **Spam** or **Ham (Not Spam)**. The project includes model training, evaluation, and a Streamlit-based web application for real-time predictions.

---

## 🚀 Features

- Detects Spam and Non-Spam emails
- Text preprocessing and cleaning
- Feature extraction using CountVectorizer
- Machine Learning-based classification
- Real-time predictions with Streamlit
- Simple and interactive user interface

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- CountVectorizer
- Streamlit
- Jupyter Notebook

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Shakshi306306/spam-email-classifier.git
cd spam-email-classifier
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

Open your browser and visit:

```text
http://localhost:8501
```
